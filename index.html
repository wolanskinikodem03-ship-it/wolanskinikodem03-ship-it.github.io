Projects

Engineering Thesis: Two-Wheeled Self-Balancing Robot

This project was developed as my engineering thesis and involved the complete design, implementation, and testing of a two-wheeled self-balancing robot. The system was based on embedded, object-oriented C++ programming and real-time control of the robot's movement.

The robot used a PID control algorithm for stabilization, PWM motor control for driving the DC motors, and sensor integration with an IMU and encoders for feedback. Bluetooth communication was also implemented to control the movement of the robot.

Key aspects: engineering thesis, embedded systems, object-oriented C++, PID control, PWM motor control, IMU integration, encoder feedback, Bluetooth control, real-time stabilization.

Autonomous Floor Cleaning Robot

This project involved designing a mobile floor cleaning robot intended for autonomous cleaning tasks. The work focused on developing the mechanical structure of the robot, including the arrangement of the main body, wheels, cleaning components, and internal space for electronic and mechanical systems.

The project required preparing technical drawings and considering the functional layout of the robot so that the design could support movement, stability, and practical use in floor cleaning applications.

Key aspects: mobile robot design, mechanical structure, technical drawings, floor cleaning mechanism, functional component arrangement.

Shaft Design for an Industrial Application

This project involved designing a shaft from scratch for an industrial application. The work included determining the shaft geometry, calculating its strength parameters, and selecting key design features such as the number of gear teeth and other mechanical elements required for the task.

Key aspects: shaft design, strength calculations, mechanical design, gear tooth selection, assembly drawing, machining drawing.
